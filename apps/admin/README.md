# Unimart - Admin Dashboard

The internal tool for sellers to manage inventory and for moderators to verify student IDs and flag inappropriate listings.

## Tech Stack
- **Framework:** Next.js
- **Charts:** Recharts (for sales data)
- **Tables:** TanStack Table

## Key Features
- **Verification Queue:** Approve/Reject student registrations.
- **Inventory Mgmt:** Bulk edit prices and availability.
- **Dispute Center:** Manage reports between buyers and sellers.

## Run Locally
```bash
pnpm dev --filter admin
