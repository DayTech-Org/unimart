# Unimart - Student Storefront

The main marketplace application where students browse, buy, and list items.

## Tech Stack
- **Framework:** Next.js (App Router)
- **UI:** `@unimart/ui` & `@unimart/ui-patterns`
- **Auth:** `@unimart/auth`
- **Data:** `@unimart/db` (Supabase)

## Getting Started
1. Ensure the root `.env` is configured.
2. Run from the project root:
   ```bash
   pnpm dev --filter web
